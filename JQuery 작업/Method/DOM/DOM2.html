삽입 메소드 part1

(A)가 기준이 됨

$(A).appendTo(B) : A를 B의 요소 내 뒷부분의 추가

$(A).prependTo(B) : A를 B의 요소 내 앞부분의 추가

$(A).insertAfter(B) : A를 B의 요소 뒤에 추가

$(A).insertBefore(B) : A를 B의 요소 앞에 추가

삽입 메소드 part2

(B)가 기준이 됨 $(A).append(B) : B를 A요소 내 뒷부분에 추가 $(A).prepend(B) : B를 A요소 내 뒷부분에 추가 $(A).after(B) : B를 A요소 내 뒷부분에 추가 $(A).before(B) : B를 A요소 내 뒷부분에 추가
